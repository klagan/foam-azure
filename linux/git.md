# git

### list the configuration settings
```
git config --list
```

## check your name
```
git config user.name
```

## set your name
```
git config --global user.name "Kam Lagan"
```

## read git configuration (~/.gitconfig)
```
git config -l
```

# list all branches

```
git branch -a
```

# switch to remote branch (locally)

```
git fetch

git checkout -t <remote_name>/<branch_name>
```

# git

### List the configuration settings
```
git config --list
```

## Check your name
```
git config user.name
```

## Set your name
```
git config --global user.name "Kam Lagan"
```

## Read git configuration (~/.gitconfig)
```
git config -l
```

## Get machine information
```
sudo lshw
```

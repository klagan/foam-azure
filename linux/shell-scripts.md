# Shell Scripts

Begin the script with:
```
#!/bin/bash
```

Set execute permission on your script: 
```
chmod +x <script>.sh
```

Execute shell in one of three ways:
```
./<script>.sh
```
```  
sh <script>.sh
```
```
bash <script>.sh
```
